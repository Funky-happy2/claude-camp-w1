# Claude Camp - Week 1 Homework

This repository contains my Week 1 project for Claude Camp. The main project is a **Greeting Generator** built in Python.

## What the Greeting Generator does

The program asks for your name and age, then generates a personalised greeting in one of **10 randomly chosen styles**:

| Style | Description |
|---|---|
| 🎩 Formal | Polite and eloquent, like a Victorian gentleman |
| 👋 Casual | Friendly and relaxed, like talking to a mate |
| 🥳 Enthusiastic | LOUD AND EXCITED ABOUT EVERYTHING!!! |
| 🌟 Poetic | Flowery and lyrical, full of metaphors |
| 🔮 Mysterious | Cryptic and oracle-like |
| ☠️ Pirate | Seafaring slang and swashbuckling energy |
| 🤖 Robot | ROBOTIC LOGIC AND ALL CAPS. BEEP BOOP. |
| 🧙 Wizard | Thee, thou, and ancient scrolls |
| 🏆 Sports Coach | Motivational and pumped up |
| 🌴 Laid-back | Chill vibes only, no rush |

Every part of the experience — the greeting itself, the save prompt, the goodbye message, and the "generate another?" question — all match the same randomly chosen style.

## Features

- **Bilingual** — supports English and Chinese (中文)
- **Save greetings** — you can save any greeting to a `Saved Greetings/` folder, stored in a file named after you with a timestamp
- **Loop** — keep generating new greetings without restarting the program

## How to run it

Make sure you have Python 3 installed, then run:

```
python "Code/Greeting Generator.py"
```
